## 📑 Description
    
**나의 백준 정보 확인**  
- 프로필 이미지, 아이디, 배지, 클래스 정보를 제공합니다.
- Solved.ac 기준 나의 실시간 티어, 점수, 랭킹을 확인할 수 있습니다.
- 내가 푼 상위 50문제를 확인할 수 있습니다.
- 내가 푼 문제 난이도 분포를 그래프, 표로 확인할 수 있습니다.

**위젯 기능**
- 작은 크기 위젯, 중간 크기 위젯을 지원합니다.
- 앱과 동일한 정보를 위젯으로 쉽게 확인할 수 있습니다.
